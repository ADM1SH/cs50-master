# CS50 Courses Master Collection

## Overview
A comprehensive repository containing coursework, problem sets, and projects from several Harvard CS50 courses. This collection demonstrates a journey through foundational computer science, pythonic programming, and full-stack web development.

## Included Courses
### 1. CS50x: Introduction to Computer Science
- Focused on low-level concepts in C, memory management, and data structures.
- Explores algorithms like merge sort and binary search.

### 2. CS50P: Introduction to Programming with Python
- Practical application of Python for automation, data analysis, and small-scale software.
- Covers libraries like `pytest`, `pandas`, and `requests`.

### 3. CS50W: Web Programming with Python and JavaScript
- Full-stack development using **Django** (Python) and **React** (JavaScript).
- Projects include building a search engine interface, an auction site, and an email client.
